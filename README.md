# hackbox-ctrl

Seed repo for a standalone control-plane toolkit.

Intended long-term structure:

- `scripts/` — tracked shared control-plane scripts
- `docs/` — tracked shared docs, specs, decisions, plans, and devlogs
- `hackbox-ctrl-inventory/` — local sibling/nested inventory checkout, intentionally untracked here

This repo is intentionally minimal for now. The extraction plan and migration
notes are being prepared from `hackbox-ctrl-inventory` and
`hackbox-ctrl-utils`.
